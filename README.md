# Starter Kit - Sites Profissionais

## Como usar

1. Clone o projeto
2. Instale dependências
3. Edite:
   - src/config/site.ts
   - src/data/*
4. Rode o projeto

## Estrutura

- components → UI e layout
- sections → blocos reutilizáveis
- data → conteúdo
- config → dados globais

## Objetivo

Criar sites modernos, rápidos e reutilizáveis para diferentes clientes.
